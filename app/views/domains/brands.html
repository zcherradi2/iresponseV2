How to updload brands :

- First of all , all brands should be in zip files , which means for every domain there is a zip file contains its landing page .
- Second thing is that zip files should be named after the domain replacing dots with underscores that you want to link to ( eg : for domain.com => domain_com.zip )
- Last but not index.html file should be renamed to home.html ( that's so important )
- PS : if you want to replace our default brand website just rename the file to default.zip
