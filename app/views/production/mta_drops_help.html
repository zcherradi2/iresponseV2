Tags Help

Main Tags :

[ip] : The Current IP
[rdns] : The Current IP's RDNS
[domain] : The Current IP's Domain
[custom_domain] : The Custom Vmta's Domain
[static_domain] : The Static Domain
[smtp_user] : Smtp Username ( Smtp Process Only )
[server] : Server Name
[email_id] : Destinator Email Id
[email] : Destinator Email
[first_name] : Destinator First Name
[last_name] : Destinator Last Name
[return_path] : Return Path
[from_name] : From Name
[subject] : Subject
[mail_date] : Current Date
[message_id] : Generated Message ID
[negative] : Negative file content ( if any )
[placeholder1] : The Current placeholder 1 value
[placeholder2] : The Current placeholder 2 value
[placeholder3] : The Current placeholder 3 value


Unique Random Tags :
(eg: [uan_12] for fixed size and [uan_5_15] for random ranged size)

[ua] : Unique Alpha Random
[ual] : Unique Alpha Lowercase Random
[uau] : Unique Alpha Uppercase Random
[uan] : Unique Alphanumeric Random
[uanl] : Unique Alphanumeric Lowercase Random
[uanu] : Unique Alphanumeric Uppercase Random
[un] : Unique Numeric Random
[uhu] : Unique Uppercase Hex Random
[uhl] : Unique Lowercase Hex Random


Random Tags :
(eg: [an_12] for fixed size and [an_5_15] for random ranged size)

[a] : Alpha Random
[al] : Alpha Lowercase Random
[au] : Alpha Uppercase Random
[an] : Alphanumeric Random
[anl] : Alphanumeric Lowercase Random
[anu] : Alphanumeric Uppercase Random
[n] : Numeric Random
[hu] : Uppercase Hex Random
[hl] : Lowercase Hex Random


Links Tags :

[open] : Open Link , eg : http://[domain]/[open]
[url] : Click Link , eg : http://[domain]/[url]
[unsub] : Unsub Link , eg : http://[domain]/[unsub]
[optout] : OptOut Link , eg : http://[domain]/[optout]


Short Links Tags : (No need for domain tag and no need for http://)

[short_open] : Open Link , eg : [short_open]
[short_url] : Click Link , eg : [short_url]
[short_unsub] : Unsub Link , eg : [short_unsub]
[short_optout] : OptOut Link , eg : [short_optout]


VMTAS Types :


Default Vmta :
Sends with default vmtas
[rdns] : will have default rdns
[domain] : will have default domain
[custom_domain] : will be empty


Custom Vmta :
Sends with custom created vmtas (IF ANY)
[rdns] : will have custom domain
[domain] : will have custom domain
[custom_domain] : will have custom domain


Merged Vmta :
Sends with custom created vmtas (IF ANY)
[rdns] : will have default rdns
[domain] : will have default domain
[custom_domain] : will have custom domain
